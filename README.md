# Claude Plugins by Max Techera

Battle-tested Claude Code plugins for content creators, founders, and developers.

## Quick Install

```bash
# Add this marketplace
claude plugin add-marketplace https://github.com/maxtechera/claude-plugins

# Install plugins
/plugin install hormozi-frameworks
/plugin install content-creator
/plugin install creator-os
```

## Available Plugins

## Plugin Details

### hormozi-frameworks

Complete business frameworks from Alex Hormozi's $100M book series:

- **Value Equation**: Dream Outcome x Likelihood / Time x Effort
- **Grand Slam Offers**: Create offers so good people feel stupid saying no
- **Core Four**: Warm outreach, cold outreach, content, paid ads
- **Rule of 100**: 100 primary actions per day until you hit your goal

**Skills included:**
- `offers` - Value Equation, pricing, packaging
- `leads` - Core Four, outreach, lead magnets
- `money-models` - LTV optimization, offer stacking
- `copywriting` - Landing page formulas, CTAs

**Commands:**
- `/find-topic` - Research winning content topics using Hormozi's framework

---

### content-creator

Complete content creation system for viral, authentic content:

- **Viral Patterns**: Hook formulas, STEPPS framework, story structures
- **Natural Writing**: AI-proof writing that sounds human
- **Multiplication**: Turn 1 piece of content into 15+ assets

**Skills included:**
- `viral-patterns` - Hooks, psychological triggers, CTAs
- `natural-writing` - Avoid AI detection, authentic voice
- `multiplication` - 1-to-15 content repurposing system

**Commands:**
- `/create-content` - Generate content with brand voice
- `/multiply-content` - Transform content into multiple formats

**Agents:**
- `content-creator` - AI agent for content generation

- **Weekly Rhythm**: Mon planning, Tue-Thu execution, Fri review
- **Sabotage Detection**: Identify self-defeating patterns
- **State Management**: Track progress with JSON state files

**Skills included:**
- `operating-system` - State file architecture
- `weekly-rhythm` - Operating cadence patterns
- `sabotage-detection` - Self-sabotage pattern recognition
- `decision-framework` - Strategic decision making

**Commands:**
- `/daily-check` - Morning review ritual
- `/weekly-plan` - Monday planning session
- `/weekly-review` - Friday metrics review
- `/sync-state` - Sync state files with reality
- `/check-sabotage` - Detect sabotage patterns

**Agents:**
- `operations-manager` - Operations and planning agent

## Manual Installation

If you prefer manual installation:

```bash
# Clone the repo
git clone https://github.com/maxtechera/claude-plugins

# Copy specific plugin to your project
cp -r claude-plugins/plugins/hormozi-frameworks .claude/plugins/

# Or copy to global plugins
cp -r claude-plugins/plugins/hormozi-frameworks ~/.claude/plugins/
```
