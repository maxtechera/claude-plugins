# Content Creator Plugin

Viral hooks and content patterns for creators.

## Skills

| Skill | Use For |
|-------|---------|
| `viral-hooks` | Write hooks that stop the scroll |

## Quick Start

```
"Write a hook for my video about [topic]"
"Score this hook using STEPPS"
"Make this title more clickable"
```

## Key Patterns

- **"How I" > "How To"** - Personal credibility wins
- **First 10 seconds** - Hook → Stakes → Credibility → Promise
- **STEPPS checklist** - Score 3+ for viral potential

## Installation

```bash
/plugin install content-creator
```
