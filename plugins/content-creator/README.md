# Content Creator Plugin

Complete content creation system with viral patterns, AI-proof writing, and multiplication.

## Skills

| Skill | Description |
|-------|-------------|
| `viral-patterns` | Hook formulas, STEPPS framework, psychological triggers |
| `natural-writing` | Eliminate AI detection patterns, human voice |
| `multiplication` | 1-to-15+ content repurposing system |

## Commands

| Command | Description |
|---------|-------------|
| `/create-content` | Generate content with templates and brand voice |
| `/multiply-content` | Transform one piece into 15+ platform-specific assets |

## Quick Start

```
"Use viral-patterns to write a hook for my video about X"
"Apply natural-writing to make this copy sound human"
"Create a multiplication plan for my latest YouTube video"
```
