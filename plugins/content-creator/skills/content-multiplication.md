# Content Multiplication Skill

The system for turning 1 piece of content into 15+ assets.

## Core Principle

```
ONE IDEA → MULTIPLE FORMATS → ALL CHANNELS → REVENUE
```

## Production Workflow

### Step 1: Create Core Content
- Record main video OR write main post
- Include: hook, stakes, credibility, solution, CTA
- Capture raw footage for clips

### Step 2: Extract Key Moments
- Identify 3-5 quotable/clipable moments
- Mark timestamps for shorts
- Note standalone insights

### Step 3: Create Platform Derivatives
For each platform:
1. Adapt format (vertical/horizontal/square)
2. Adjust length (15s/30s/60s/long)
3. Customize CTA for platform
4. Apply platform-specific voice

### Step 4: Schedule Distribution

| Day | Platform | Content |
|-----|----------|---------|
| Monday | YouTube | Long-form publish |
| Tuesday | Newsletter | NODO edition |
| Tue-Thu | Instagram | Reels + Stories |
| Wed-Fri | Twitter | Thread + clips |
| Thursday | LinkedIn | Professional post |

## Repurposing Rules

1. **Never just repost** - Adapt for each platform
2. **Lead with value** - Not "check my YouTube"
3. **Native format** - Vertical for IG/TikTok, horizontal for YouTube
4. **Platform voice** - LinkedIn more professional, Instagram more casual
5. **Stagger timing** - Don't flood all channels same day
