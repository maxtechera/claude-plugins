## The "Hook, Retain, Reward" Model

Hormozi's master framework for viral content:

```
HOOK (First 3 seconds)
├── Pattern interrupt: Bold statement, surprising stat, pain-point
├── 50% watch without audio - text on screen essential
├── Goal: Stop the scroll
└── Use "How I" for credibility

RETAIN (Middle 3-45 seconds)
├── Story structure: Tease → Build → Reveal
├── Don't dive straight into lesson
├── Neural coupling: Brain syncs with storyteller
└── Keep teasing the main point

REWARD (End)
├── Deliver massive value
├── Clear, actionable takeaway
└── Strong CTA
```

## The "How I" Credibility Rule

> **"How I" beats "How To" every time.**

| ❌ Generic "How To" | ✅ Credible "How I" |
|---------------------|---------------------|
| How to get more leads | How I got 1,000 leads in 30 days |
| How to automate | How I automated 80% of my workflow |
| The best prompts | The prompts I use every day |
| Tips for productivity | My system that saves me 10 hours/week |

### Why This Works

1. **Requires receipts** - You can't say "How I" without proof
2. **Establishes authority** - You did it, not just read about it
3. **Creates specificity** - Forces real numbers, real timelines
4. **Builds trust** - Personal experience > generic advice

### The Formula

```
"How I" + [specific result] + [timeframe] + [constraint overcome]
```

**Example:**
- ❌ "How to use Claude Code effectively"
- ✅ "How I cut my dev time from 3 days to 3 hours using Claude Code"

### When Teaching (Workshops, Courses)

Keep personal framing ("My rule", "What I use") but ADD their benefit:

```
❌ "My rule that I use every day"
✅ "My rule that I use every day - and that's going to save you hours"

❌ "The 4 prompts I use"
✅ "The 4 prompts I use for almost everything. Copy them. They work."
```

**The pattern:** Personal credibility FIRST → Their benefit SECOND

## Proven Hook Formulas

### Hormozi Value Equation
```
Score = (Dream Outcome x Likelihood) / (Time x Effort)
Target: Score 8+
```

**Scoring Guide**:
- Dream Outcome (1-10): How big is the transformation?
- Likelihood (1-10): How believable is success?
- Time (1-10, inverted): How fast? (10=instant)
- Effort (1-10, inverted): How easy? (10=effortless)

### The 6 Proven Hook Templates

| Type | Formula | Example |
|------|---------|---------|
| **Money Hook** | "I spent $[X] on [thing]..." | "I spent $50K on ads - here's what I learned" |
| **Counterintuitive** | "Why [positive people] stay [negative]" | "Why smart founders stay broke" |
| **Journey Hook** | "How I went from [A] to [B] in [time]" | "How I went from $0 to $1M in 18 months" |
| **Failure Hook** | "The $[X] mistake that almost..." | "The $200K mistake that almost bankrupted me" |
| **Secret Hook** | "What [experts] don't tell you..." | "What gurus don't tell you about passive income" |
| **How I Hook** | "How I [result] without [method]" | "How I got 100K followers without posting daily" |

### Hook Categories by Performance

**Money/Results Hooks (3.2x engagement)**:
- "Gaste $[X] en [tool] - Los resultados te van a shockear"
- "Como [logre resultado] sin gastar un peso"
- "De $0 a $[X] con [metodo]"
- "Este [metodo] genero $[X] en [tiempo]"

**Transformation Hooks (2.8x engagement)**:
- "De Uruguay a Silicon Valley: La verdad que nadie cuenta"
- "Como pase de [A] a [B] en [tiempo]"
- "[X] años atras estaba [mal]. Hoy [resultado]"
- "El journey de [origen] a [destino] (paso a paso)"

**Counterintuitive Hooks (2.5x engagement)**:
- "Por que [declaracion contraintuitiva] es en realidad [verdad]"
- "[Cosa popular] esta MUERTO - Esto lo reemplaza"
- "Todos dicen [X]. Estan equivocados..."
- "[Experto] me dijo [consejo]. Hice lo opuesto y [resultado]"

**Failure Hooks (2.3x engagement)**:
- "El error de $[X] que casi me cuesta todo"
- "Perdi [X] por no saber esto"
- "Lo que me hubiera gustado saber antes de [accion]"
- "El mistake que destruyo mi [cosa]"

**Secret/Insider Hooks (2.1x engagement)**:
- "El truco que [grupo] no quiere que sepas"
- "Lo que [expertos] no te cuentan sobre [tema]"
- "La estrategia oculta que nadie menciona"
- "Por que [industria] no quiere que sepas esto"

### Hook Optimization Example

```
❌ Before: "How to grow on Instagram"
   Score: (5 × 3) / (8 × 8) = 0.23

✅ After: "How I grew from 0 to 10K followers in 30 days without posting daily"
   Score: (9 × 7) / (3 × 2) = 10.5

What changed:
- "How I" → adds proof (likelihood)
- "0 to 10K" → specific dream outcome
- "30 days" → reduces time
- "without posting daily" → reduces effort
```

## Psychological Triggers (Cialdini's 7 + STEPPS)

### Cialdini's 7 Principles
| Principle | Application |
|-----------|-------------|
| **Reciprocity** | Give massive free value first |
| **Commitment** | Small ask → bigger ask later |
| **Social Proof** | "Join X,000+ others", testimonials |
| **Authority** | Credentials, specific metrics |
| **Liking** | Personal stories, relatability |
| **Scarcity** | Limited time/spots |
| **Unity** | "We" language, shared identity |

### Trigger Templates
- **Reciprocity**: "Estoy regalando [cosa valiosa]. Sin catch."
- **Social Proof**: "[X,000] personas ya usan esto para [resultado]..."
- **Scarcity**: "Esto no va a estar disponible despues de [deadline]..."
- **Unity**: "Si sos [identidad compartida], esto es para nosotros..."

## Visual Identity (Hormozi Pattern)

### Thumbnail Formula

1. **Face with emotion** (shocked/intrigued) = highest CTR
2. **Red arrows** ONLY when pointing to key element
3. **Yellow/Red text** on dark background
4. **Before/after** split screens work
5. **Money visuals + face** = trust + curiosity
6. **3-4 words maximum**

### Caption Style

- **Font**: Anton (or bold sans-serif)
- **Case**: UPPERCASE
- **Effects**: Shadow + yellow stroke
- **Animation**: Pop-in from bottom
- **Accents**: Emoticons for topics, color changes for keywords

**50% of viewers watch without audio** - captions are essential.

## STEPPS Viral Checklist

Score 3+/6 for viral potential:

| Letter | Principle | Question | Impact |
|--------|-----------|----------|--------|
| **S** | Social Currency | Makes them look smart sharing? | +40% shares |
| **T** | Triggers | Connects to something they see daily? | +30% recall |
| **E** | Emotion | Generates admiration, surprise, or FOMO? | +60% shares |
| **P** | Public | Visible when implemented? | +25% adoption |
| **P** | Practical Value | Applicable TODAY? | +35% saves |
| **S** | Stories | Has memorable narrative? | +50% retention |

### Scoring Example
```
Content: "How I automated 80% of my workflow using AI"

[x] Social Currency: Makes sharer look tech-savvy
[x] Triggers: People think about AI daily
[x] Emotion: FOMO, desire, admiration
[ ] Public: Partially visible
[x] Practical Value: Immediately actionable
[x] Stories: Personal journey embedded

Score: 5/6 - HIGH VIRAL POTENTIAL
```

## Open Loops (Zeigarnik Effect)

**Psychology**: Incomplete tasks create mental tension. Brain needs closure.

### Templates
```
"...pero el tercero cambio todo"
"Antes de contarte [cosa principal]..."
"Quedate hasta el final para..."
"Hay algo que no mencione todavia..."
"Aca es donde se pone interesante..."
"Lo que paso despues me sorprendio..."
```

### Placement
- **Hook**: Create initial curiosity
- **After first point**: Keep watching
- **Mid-content**: Prevent drop-off
- **Pre-CTA**: Motivate action

## Story Structures

### Before-After-Bridge (BAB)
```
BEFORE: "Estas [sufriendo problema]. Sentis [emocion]..."
AFTER: "Imagina en cambio [escenario ideal]. [Resultado positivo]..."
BRIDGE: "El puente es [tu metodo]. Asi funciona..."
```

### Problem-Agitate-Solve (PAS)
```
PROBLEM: "[Audiencia], estas luchando con [problema]..."
AGITATE: "Y te esta costando [tiempo/plata/status]. Cada dia que esperas..."
SOLVE: "Hay una mejor manera. [Solucion] te permite [resultado]..."
```
