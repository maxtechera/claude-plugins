# Creator OS Plugin

Personal operating system for high-output creators with state management and sabotage detection.

## Skills

| Skill | Description |
|-------|-------------|
| `operating-system` | State file architecture, tracking systems |
| `weekly-rhythm` | Monday planning, execution days, Friday review |
| `sabotage-detection` | Detect patterns that prevent shipping |

## Commands

| Command | Description |
|---------|-------------|
| `/daily-check` | Morning state review, priorities, blockers |
| `/weekly-plan` | Monday planning ritual, define Big 3 |
| `/check-sabotage` | Full sabotage pattern analysis |

## Quick Start

```
"Run /daily-check to see my current state"
"Help me with /weekly-plan for this week"
"Run /check-sabotage to see if I'm self-sabotaging"
```

## Key Frameworks

### Weekly Rhythm
- **Monday**: Planning - Define Big 3
- **Tuesday-Thursday**: Execution - Ship > Polish
- **Friday**: Review - Metrics, learnings, one adjustment

### Big 3 Format
1. Norte dominante (main focus)
2. Satelite 1 (supporting)
3. Satelite 2 (supporting)

### 7 Sabotage Patterns
1. Dispersion Noble - Too many things in progress
2. Dopamina de Inicio - Addicted to starting
3. Perfeccionismo - Polish instead of ship
4. Optimizacion Prematura - Tools before output
5. Negociar con Vos Mismo - Changing plans mid-week
6. Ansiedad de Oportunidad - Can't say no
7. Rumiacion Productiva - Planning without doing

## Installation

```bash
/plugin install creator-os
```
