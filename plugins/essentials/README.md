# Essentials Plugin

Reference guides for extending Claude Code with custom skills, agents, commands, and MCP servers.

## Skills

| Skill | Description |
|-------|-------------|
| `skills-guide` | How to create and structure skill files |
| `agents-guide` | How to define custom agents (subagent_type) |
| `commands-guide` | How to create slash commands |
| `mcp-guide` | How to configure and create MCP servers |

## Usage

```
"How do I create a custom skill?"
"Show me how to define a custom agent"
"What's the format for slash commands?"
"Help me set up an MCP server"
```

## What You'll Learn

### Skills
- Skill file format (.md structure)
- Frontmatter and metadata
- Best practices for effective skills
- Skill vs prompt differences

### Agents
- Custom agent type definitions
- Tool access configuration
- Agent prompts and behavior
- Task tool integration

### Commands
- Slash command file format
- Command discovery and registration
- Arguments and parameters
- User-invocable commands

### MCP Servers
- Server configuration format
- Transport types (stdio, SSE)
- Custom server creation
- Tool exposure and permissions

## Installation

```bash
/plugin install essentials@max-techera-plugins
```
