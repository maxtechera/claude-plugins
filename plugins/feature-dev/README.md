# Feature Dev Plugin

Structured 7-phase workflow for building features properly.

## Command

| Command | Description |
|---------|-------------|
| `/feature-dev` | Guided feature development with exploration and review |

## The 7 Phases

1. **Discovery** - Clarify requirements
2. **Exploration** - Understand codebase with parallel agents
3. **Questions** - Fill gaps before designing
4. **Architecture** - Present approaches, get approval
5. **Implementation** - Build with conventions
6. **Review** - Parallel agents check quality
7. **Summary** - Document and suggest next steps

## Quick Start

```
/feature-dev Add user authentication with OAuth
```

Or just:
```
/feature-dev
```

## Philosophy

- Understand before changing
- Ask before assuming
- Get approval before building
- Review before finishing

## Installation

```bash
/plugin install feature-dev@max-techera-plugins
```
