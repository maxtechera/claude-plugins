# Hormozi Frameworks Plugin

Complete business frameworks from Alex Hormozi's $100M book series.

## Skills

| Skill | Description |
|-------|-------------|
| `offers` | Value Equation, Grand Slam Offers, pricing, guarantees |
| `leads` | Core Four, Rule of 100, lead magnets, outreach |
| `money-models` | Three Stages, upsells/downsells, LTV optimization |
| `copywriting` | Landing page formulas, CTAs, psychological triggers |

## Commands

| Command | Description |
|---------|-------------|
| `/find-topic` | Find winning content topics using the Hormozi framework |

## Quick Start

After installing, use skills by referencing them in your prompts:

```
"Use the offers skill to create a value proposition for my course"
"Apply the leads framework to plan my content strategy"
"Help me design an offer stack using money-models"
```

## Key Frameworks

### Value Equation
```
Value = (Dream Outcome x Perceived Likelihood) / (Time Delay x Effort)
```
Target score: 8+

### Core Four (Lead Generation)
1. Warm Outreach
2. Cold Outreach
3. Free Content
4. Paid Ads

### Three Stages (Revenue)
1. Get Cash (Attraction offers)
2. Get More Cash (Upsells/Downsells)
3. Get Most Cash (Continuity)

## Installation

```bash
/plugin install hormozi-frameworks
```
