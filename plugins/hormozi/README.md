# Hormozi Plugin

Value Equation from $100M Offers for pricing and positioning.

## Skills

| Skill | Use For |
|-------|---------|
| `value-equation` | Price and position offers |

## Quick Start

```
"Score my offer using the value equation"
"How can I make this offer more valuable?"
"Audit my landing page with the value equation"
```

## The Formula

```
Value = (Dream Outcome × Likelihood) / (Time × Effort)
```

- Increase Dream Outcome (what they really want)
- Increase Likelihood (proof it works)
- Decrease Time (faster results)
- Decrease Effort (easier for them)

## Installation

```bash
/plugin install hormozi
```
