# MailerLite Plugin

Complete MailerLite integration for email marketing, automations, and API operations.

## Skills

| Skill | Description |
|-------|-------------|
| `mailerlite-mastery` | Complete API reference, automation patterns, email templates |

## Usage

```
"Create a welcome email sequence"
"Add subscriber to MailerLite with these fields"
"Set up a re-engagement automation"
"Generate HTML email template"
"Explain MailerLite webhook setup"
```

## Features

- **API Reference**: All endpoints with examples
- **Automation Patterns**: Welcome, lead magnet, re-engagement, e-commerce
- **Email Templates**: Proven HTML templates (61% open rate)
- **Webhook Integration**: Event handling with code examples
- **Segmentation**: Groups vs segments, proven strategies

## Requirements

- MailerLite API key (in environment variables)
- `MAILERLITE_API_KEY` or `MAILERLITE_API_TOKEN`

## Installation

```bash
/plugin install mailerlite@max-techera-plugins
```
