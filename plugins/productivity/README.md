# Productivity Plugin

Sabotage pattern detection to stay focused and ship.

## Skills

| Skill | Use For |
|-------|---------|
| `sabotage-check` | Detect patterns killing your productivity |

## Quick Start

```
"Am I sabotaging myself right now?"
"Run a sabotage check on my current work"
"Why haven't I shipped anything this week?"
```

## The 4 Patterns

1. **Noble Dispersion** - Too many things open, nothing finished
2. **Start Dopamine** - Starting > Finishing
3. **Perfectionism** - Polishing > Shipping
4. **Premature Optimization** - Tools > Output

## Installation

```bash
/plugin install productivity
```
