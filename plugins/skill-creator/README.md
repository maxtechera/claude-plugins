# Skill Creator Plugin

Meta-skill for creating new Claude Code skills.

## Skill

| Skill | Use For |
|-------|---------|
| `skill-creator` | Building new skills with best practices |

## Quick Start

```
"Help me create a skill for API response formatting"
"I want to build a skill for our database migrations"
```

## What You'll Learn

- Skill anatomy (SKILL.md + optional scripts/references)
- Frontmatter format for auto-activation
- Progressive disclosure patterns
- Best practices for concise, effective skills

## Key Concepts

1. **Be Concise** - Only add what Claude doesn't know
2. **Progressive Disclosure** - Layer information appropriately
3. **Clear Triggers** - Good descriptions = proper activation
4. **Freedom Levels** - Match constraint level to task criticality

## Installation

```bash
/plugin install skill-creator@max-techera-plugins
```
