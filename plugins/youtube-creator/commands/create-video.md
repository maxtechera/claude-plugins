# Create Video Command

Execute the full video production pipeline from idea to multiplication.

## Process

### Phase 1: Research & Validation

1. **Topic Validation**
   - Is this topic validated? (STEPPS score 3+/6)
   - Do you have credibility/receipts for this topic?
   - What's the "How I" angle?

2. **Competitive Research**
   - What videos exist on this topic?
   - What's missing from existing content?
   - What's your unique angle?

3. **Audience Questions**
   - What questions does your audience have about this?
   - What objections might they have?
   - What's the transformation they want?

### Phase 2: Planning

1. **Define Video Metadata**
   ```
   Title: [Working title]
   Target Length: [X] minutes
   Target Audience: [Specific description]
   Main Takeaway: [One sentence]
   Success Metric: [Views/Leads/Conversions]
   ```

2. **Outline Main Points**
   - Point 1: [Foundation concept]
   - Point 2: [Build on foundation]
   - Point 3: [Advanced insight/transformation]

3. **Identify Stories/Examples**
   - Personal story for Point 1
   - Case study for Point 2
   - Proof/receipts for Point 3

### Phase 3: Script Writing

Generate full script using the youtube-scripting skill:

```
## HOOK (30 seconds)
[Pattern interrupt → Stakes → Credibility → Promise]

## POINT 1: [Title]
[Setup → Story → Takeaway → Transition]

## POINT 2: [Title]
[Setup → Story → Takeaway → Transition]

## POINT 3: [Title]
[Setup → Story → Takeaway → Transition]

## CONCLUSION
[Recap → CTA → Teaser]
```

### Phase 4: Production Prep

1. **B-Roll List**
   - [Timestamp]: [What to capture/find]
   - [Timestamp]: [What to capture/find]

2. **Graphics/Text Overlays**
   - [Timestamp]: [Text to display]
   - [Timestamp]: [Graphic needed]

3. **Thumbnail Concept**
   - Emotion: [What expression]
   - Text: [3-4 words]
   - Visual: [Supporting element]

4. **Title Options** (3 variations)
   - Option A: [Title]
   - Option B: [Title]
   - Option C: [Title]

### Phase 5: Recording Checklist

Before recording:
- [ ] Script practiced (not memorized word-for-word)
- [ ] Energy warmed up
- [ ] Camera/audio tested
- [ ] Lighting set
- [ ] B-roll list visible
- [ ] Water nearby

### Phase 6: Multiplication Plan

After video is recorded/published:

**YouTube Shorts** (3-5 clips)
- [ ] Clip 1: [Hook moment]
- [ ] Clip 2: [Key insight]
- [ ] Clip 3: [Practical tip]

**Instagram Reels** (2-3 clips)
- [ ] Reel 1: [Condensed story]
- [ ] Reel 2: [Tutorial snippet]

**Other Platforms**
- [ ] Twitter thread
- [ ] LinkedIn post
- [ ] Blog post
- [ ] Newsletter feature
