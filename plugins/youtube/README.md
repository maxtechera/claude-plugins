# YouTube Plugin

YouTube scripting and video structure templates.

## Skills

| Skill | Use For |
|-------|---------|
| `scripting` | Write engaging video scripts |

## Quick Start

```
"Write a script for a video about [topic]"
"Structure my video using the hook-intro-body-close format"
"Help me write a hook for this tutorial"
```

## Video Structure

1. **Hook** (0-10s) - Stop the scroll
2. **Intro** (10-30s) - Promise value
3. **Body** - Story → Lesson → Action
4. **Close** - Recap + CTA + Tease

## Installation

```bash
/plugin install youtube
```
