# YouTube Scripting Skill

Templates and formulas for YouTube content.

## Video Structure Template

### Long-Form (10-15 min)

```markdown
## HOOK (0-10 seconds)
[Pattern interrupt - grab attention immediately]
"Che, gaste $800 en Claude Code este mes..."

## STAKES (10-20 seconds)
[Why should they care?]
"...y los resultados cambiaron todo mi negocio"

## CREDIBILITY (20-30 seconds)
[Why should they listen to YOU?]
"Como CTO en Silicon Valley con 8 anos de experiencia..."

## PROMISE (30-45 seconds)
[What will they learn?]
"Hoy te voy a mostrar exactamente como configurar tu setup"

## CONTENT SECTIONS (bulk of video)
### Section 1: [Topic]
- Key point
- Example/demo
- Transition

### Section 2: [Topic]
- Key point
- Example/demo
- Transition

### Section 3: [Topic]
- Key point
- Example/demo
- Transition

## RECAP (30 seconds)
[Summarize key takeaways]

## CTA (15 seconds)
[One clear action]
"Si te sirvio, suscribite al canal y activa la campanita"
```

### Shorts (15-60 seconds)

```markdown
## HOOK (0-3 seconds)
[Immediate pattern interrupt]
"STOP - esto es lo que nadie te dice sobre Claude Code"

## ONE KEY POINT (3-45 seconds)
[Single focused insight with example]

## CTA (last 5 seconds)
[Quick action]
"Follow para mas tips"
```

## Script Writing Rules

### Hook Formula
Use Hormozi Value Equation:
```
(Dream Outcome x Likelihood) / (Time x Effort) = Value Score
Target: 8+
```

### Engagement Tactics
- Ask questions: "Alguna vez te paso que...?"
- Pattern interrupts every 30-60 seconds
- Visual changes every 5-10 seconds (in editing)
- Open loops: "Pero antes de eso..."

### Language Guidelines
- 80% Spanish, 20% English (technical terms)
- Short sentences (under 15 words)
- Conversational tone ("Mira", "Che", "O sea")
- Avoid jargon unless explaining it

## Tutorial Video Template

```markdown
## WHAT WE'RE BUILDING (30 sec)
[Show the end result first]

## WHY THIS MATTERS (30 sec)
[Personal story or pain point]

## PREREQUISITES (15 sec)
[Quick list of what they need]

## STEP-BY-STEP
### Step 1: [Action]
- Show the action
- Explain why
- Common mistakes to avoid

### Step 2: [Action]
[Repeat pattern]

## COMMON ISSUES (60 sec)
[Top 3 problems and solutions]

## RESULTS (30 sec)
[Show working demo]

## NEXT STEPS (15 sec)
[What to learn next + CTA]
```

## Thumbnail Considerations

When scripting, note:
- Key visual moment for thumbnail
- Quotable text overlay
- Emotion to capture (shock, curiosity, excitement)
